# MenuMeters

MenuMeters for macOS Tahoe (El Capitan minimum). Forked from: https://github.com/yujitach/MenuMeters/

![MenuMeters screenshot](menumeters-leeliu.png)

## What's new in this fork (v2.2.0)

- Add top processes sections to network and memory dropdown menus
- Updated network (MB instead of MB/s) and memory labels (GB instead of MB) to save space
- Change defaults for CPU, memory, network, and disk meters
- Add preference sliders for memory and network top process counts
- Clean up dropdown menus

# Usage:

Download the latest release from https://github.com/leeliu/MenuMeters/releases.

# Background:

It's a great utility originally developed at http://www.ragingmenace.com/software/menumeters/ .
The original version does not work on El Capitan and later.
